# jsapi-documentation
[![Dependency Status](https://gemnasium.com/softgames/jsapi-documentation.png)](https://gemnasium.com/softgames/jsapi-documentation)

### Requirements
* node.js 0.8
* npm 1.1

### Commands
#### Install
```bash
npm install
```
#### Start
```bash
npm start
```
#### Run the tests
```bash
npm test
```
